# CAT Digital — Portfolio Website

A modern, animated, responsive portfolio website for **CAT Digital**.

## Run locally

1. Install dependencies:
   - `npm install`
2. Start dev server:
   - `npm run dev`
3. Build for production:
   - `npm run build`
4. Preview production build:
   - `npm run preview`

## Customize content

All content lives in `src/content/siteContent.ts` (services, projects placeholders, testimonials, FAQ, etc.).
